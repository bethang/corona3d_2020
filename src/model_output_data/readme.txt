Readme file for model_output_data.
Model output data for "Nonthermal hydrogen loss at Mars: Contributions of photochemical mechanisms to escape and identification of key processes"
Authors: Bethan Gregory, Michael Chaffin, Rodney Elliott, Justin  Deighan, Hannes Groeller, and Eryn Cangi.

- "escape_probabilities_LSA" and "escape_probabilities_HSA" give the model-generated escape probabilities at a series of altitude for 5 energies, for low (LSA) and high (HSA) solar activity.
